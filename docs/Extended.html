void SPG_FloodFill8(SDL_Surface *dst, Sint16 x, Sint16 y, Uint32 color)
- Fills all 8-way (adjacent and DIAGONAL) adjoining pixels of the same color as the pixel at (x, y) with the given color.  Useful for changing the color of primitives and borders.
