SPG_bool This is a define that makes the SPG_bool type identical to Uint8.  Booleans in C++  (and any other language) are 8 bits in memory for architecture reasons, despite the fact that they hold only 1 bit of information.
SPG_DirtyTable {
Uint16 size;
SDL_Rect* rects;
Uint16 count;
Uint16 best;
}

This is a struct to hold dirty rect information.  When you retrieve the front table using SPG_DirtyUpdate(), you should loop over the table to replace the background image.

Example:
  SPG_DirtyTable* table = SPG_DirtyUpdate(screen);
  int i;
  for(i = 0; i < table->count; i++)
  {
  // Blit background to table->rects[i]
  SDL_BlitSurface(mybackground, &bgrect, screen, &(table->rects[i]);
  }
SPG_Point {

A struct to hold coordinate data (position, velocity, etc.).  It can be initialized like so:
SPG_Point mypoint = {30, 40};

mypoint = SPG_MakePoint(30, 40);
